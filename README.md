## 使用的学习方法（暂定）：

1. 先跟着把代码都敲一遍，边敲边查找不会或不熟悉的知识点
2. 在第一步的基础上，尝试自己实现一遍，并记录下容易卡壳的点
3. 第三步在原代码和自己写的代码上进行比较，进一步查缺补漏
4. 过一段时间后，再自己完全实现一遍，加以巩固
